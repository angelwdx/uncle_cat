# 猫叔智能小说创作助手使用指南

## 1. 项目介绍

猫叔智能小说创作助手（原名DeepStory AI）是一个基于AI驱动的智能小说创作平台，旨在帮助作者完成从创意构思到章节写作的全流程创作。该平台集成了多种AI模型，提供了完整的小说创作工作流，包括核心架构生成、角色设计、世界观构建、情节架构和章节写作等功能。

### 1.1 核心功能亮点

- **结构化创作流程**：引导用户从初始创意到完整小说的八步创作流程
- **多AI模型支持**：兼容Google Gemini、OpenAI、Claude、DeepSeek等多种AI提供商
- **智能状态管理**：自动同步全局摘要与角色状态，保持剧情一致性
- **丰富的可视化展示**：支持Markdown渲染、DNA可视化、世界观可视化
- **灵活的自定义选项**：支持自定义提示词、模板和创作要求
- **项目管理功能**：支持项目的导入、导出和自动保存

### 1.2 技术栈

- 前端框架：React 19
- 类型系统：TypeScript
- 构建工具：Vite
- 样式框架：Tailwind CSS
- UI组件：lucide-react图标库
- API集成：支持多种AI模型接口
- 状态管理：React内置状态管理

## 2. 安装与配置

### 2.1 环境要求

- Node.js 18+（推荐使用最新LTS版本）
- npm或yarn包管理器
- 现代浏览器（Chrome、Firefox、Edge等）

### 2.2 本地开发

1. **克隆仓库**
   ```bash
   git clone https://github.com/angelwdx/uncle_cat.git
   cd uncle_cat
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **访问应用**
   打开浏览器访问 `http://localhost:5173`

### 2.3 构建部署

1. **生产构建**
   ```bash
   npm run build
   ```

2. **构建带隐藏提示词管理的版本**
   ```bash
   npm run build:hide-prompts
   ```

3. **预览构建结果**
   ```bash
   npm run preview
   ```

## 3. 快速开始

### 3.1 初始设置

1. **打开应用**
   访问应用URL（本地开发：`http://localhost:5173`）

2. **配置AI模型**
   - 点击右上角设置图标
   - 选择AI提供商（默认：Google）
   - 输入API密钥
   - 选择模型名称
   - 保存配置

3. **开始创作**
   - 点击左侧菜单中的"创作初始化"
   - 填写小说基本信息
   - 点击"下一步"进入核心DNA生成

### 3.2 创作流程概览

| 步骤 | 名称 | 功能 |
|------|------|------|
| 1 | 创作初始化 | 设置小说基本信息 |
| 2 | 核心DNA | 生成故事核心公式 |
| 3 | 角色动力学 | 设计角色设定和关系 |
| 4 | 世界观 | 构建故事背景 |
| 5 | 情节架构 | 设计剧情结构 |
| 6 | 章节蓝图 | 生成章节规划 |
| 7 | 角色状态库 | 建立初始角色状态 |
| 8 | 正文创作 | 逐章生成小说内容 |

## 4. 详细功能说明

### 4.1 创作初始化

**功能**：设置小说的基本信息和创作参数

**操作步骤**：
1. 填写故事主题
2. 选择故事题材
3. 设置故事基调
4. 选择结局倾向
5. 设置叙事视角
6. 输入预计章节数
7. 设置每章字数
8. 填写自定义要求
9. 填写小说名称

**注意事项**：
- 主题和题材是必填项
- 自定义要求可以为空
- 可以随时返回修改这些设置

### 4.2 核心DNA

**功能**：生成故事的核心公式，定义故事的基本框架

**操作步骤**：
1. 点击"生成"按钮
2. 等待AI生成核心DNA
3. 查看生成结果
4. 可以选择修改或重新生成
5. 可以使用"判官"功能获取改进建议

**判官功能**：
- 点击"判官"按钮
- 等待AI评审
- 查看评审结果和改进方案
- 选择一个方案重写核心DNA

### 4.3 角色动力学

**功能**：设计具有复杂关系和成长弧线的角色群

**操作步骤**：
1. 点击"生成"按钮
2. 等待AI生成角色设定
3. 查看生成的角色列表和关系
4. 可以选择修改或重新生成

**生成内容**：
- 主要角色的基本信息
- 角色关系网络
- 角色成长弧线
- 角色动机和冲突

### 4.4 世界观

**功能**：构建支撑故事发展的三维世界体系

**操作步骤**：
1. 点击"生成"按钮
2. 等待AI生成世界观设定
3. 查看生成的世界观内容
4. 可以选择修改或重新生成

**生成内容**：
- 世界的物理规则
- 社会结构和文化
- 历史背景和关键事件
- 地理环境和地点

### 4.5 情节架构

**功能**：设计整体剧情结构

**操作步骤**：
1. 点击"选择剧情结构"按钮
2. 选择合适的剧情结构模板
3. 点击"生成"按钮
4. 等待AI生成情节架构
5. 查看生成结果

**支持的剧情结构**：
- 三幕式结构
- 英雄之旅
- 五幕式结构
- 非线性结构

### 4.6 章节蓝图

**功能**：生成详细的章节规划，包括每章的定位、作用和悬念设计

**操作步骤**：
1. 点击"生成"按钮
2. 等待AI生成章节蓝图
3. 查看详细的章节规划
4. 可以选择修改或重新生成

**生成内容**：
- 每章的标题
- 核心作用
- 角色发展
- 悬念设计
- 情节推进

### 4.7 角色状态库

**功能**：管理角色的动态状态和发展

**操作步骤**：
1. 点击"生成"按钮
2. 等待AI生成初始角色状态
3. 查看生成的角色状态
4. 可以查看状态历史记录

**状态内容**：
- 角色的当前状态
- 角色关系变化
- 角色成长轨迹
- 关键事件影响

### 4.8 正文创作

**功能**：逐章生成小说内容

**操作步骤**：
1. 选择要创作的章节
2. 点击"生成"按钮
3. 等待AI生成章节内容
4. 查看生成的章节
5. 可以直接编辑内容
6. 点击"同步上下文"更新全局状态

**创作工具**：
- 章节生成
- 章节重写
- 标题编辑
- 内容编辑
- 上下文同步

## 5. 高级功能

### 5.1 AI模型管理

**功能**：配置和切换不同的AI模型

**操作步骤**：
1. 点击右上角设置图标
2. 在配置模态框中选择AI提供商
3. 输入API密钥
4. 选择模型名称
5. 保存配置

**支持的AI提供商**：
- Google Gemini
- OpenAI
- Claude
- DeepSeek
- 自定义模型

### 5.2 提示词管理

**功能**：查看、编辑和管理生成提示词

**操作步骤**：
1. 点击右上角提示词管理图标
2. 在提示词管理模态框中选择要编辑的提示词
3. 编辑提示词内容
4. 保存修改

**注意事项**：
- 提示词管理功能可能在部分构建版本中隐藏
- 修改提示词会影响AI生成的内容质量
- 建议在修改前备份原始提示词

### 5.3 项目管理

**功能**：导入、导出和管理创作项目

**操作步骤**：

#### 导出项目
1. 点击右上角导出图标
2. 选择保存位置
3. 保存项目文件（.json格式）

#### 导入项目
1. 点击右上角导入图标
2. 选择要导入的项目文件
3. 等待项目加载完成

**注意事项**：
- 导出的项目文件包含所有创作数据
- 导入项目会覆盖当前的创作进度
- 建议定期导出项目以备份数据

### 5.4 自定义请求

**功能**：向AI发送自定义请求，获取特定内容

**操作步骤**：
1. 在任何步骤点击"自定义"按钮
2. 在自定义请求模态框中输入请求内容
3. 点击"发送"按钮
4. 查看AI响应

**使用场景**：
- 修改特定内容
- 生成额外的背景资料
- 获取创作建议
- 解决创作瓶颈

## 6. 创作技巧与最佳实践

### 6.1 提高AI生成质量的技巧

1. **明确的提示词**：提供清晰、具体的创作要求
2. **逐步细化**：从核心设定到具体章节，逐步细化要求
3. **利用反馈循环**：多次生成和修改，不断优化结果
4. **结合人类创意**：AI生成作为辅助，结合人类的创意和判断
5. **保持一致性**：使用同步上下文功能，保持剧情一致性

### 6.2 常见创作问题解决

1. **角色形象不够鲜明**
   - 增加角色的具体细节
   - 为角色添加独特的习惯或口头禅
   - 明确角色的动机和目标

2. **剧情发展不合理**
   - 调整情节架构
   - 检查角色行为是否符合设定
   - 确保情节有足够的铺垫和转折

3. **世界观不够完整**
   - 增加世界观的细节和规则
   - 确保世界观内部逻辑一致
   - 展示世界观对角色和情节的影响

4. **章节内容质量不高**
   - 优化提示词
   - 调整AI模型或参数
   - 手动编辑和润色

### 6.3 移动端使用技巧

- 使用侧边栏导航切换创作步骤
- 利用触摸友好的交互设计
- 注意移动端的屏幕尺寸限制
- 定期同步上下文，确保状态更新

## 7. 常见问题解答

### 7.1 连接问题

**Q：AI生成失败，提示"API调用失败"**
A：请检查以下几点：
- API密钥是否正确
- 网络连接是否正常
- AI提供商服务是否正常
- 防火墙或代理设置是否影响连接

**Q：生成内容为空或不完整**
A：可能是由于：
- 提示词不够清晰或具体
- AI模型生成超时
- API配额限制
- 网络不稳定

### 7.2 功能问题

**Q：提示词管理功能不见了**
A：提示词管理功能可能在部分构建版本中隐藏。可以尝试使用完整构建版本，或重新构建项目时不使用`build:hide-prompts`命令。

**Q：无法同步上下文**
A：请确保：
- 当前章节有内容
- AI模型配置正确
- 网络连接正常

**Q：项目导入失败**
A：请检查：
- 项目文件格式是否正确
- 文件是否损坏
- 项目版本是否兼容

### 7.3 性能问题

**Q：应用运行缓慢**
A：建议：
- 关闭其他占用资源的浏览器标签页
- 清除浏览器缓存
- 更新浏览器到最新版本
- 减少同时运行的应用程序

**Q：生成内容速度慢**
A：可能是由于：
- AI模型响应较慢
- 网络连接不稳定
- 生成内容较长
- 服务器负载较高
